# node-api-auth-jwt

## Node JS API Authentication with JWT and Bcrypt JS

## Clone Repo

```bash
git clone https://github.com/leelaing/node-api-auth-jwt.git (new-name)
```

## Install Node Modules

```bash
npm i
```

or

```bash
yarn
```

## Rename .env.example

Rename to .env and complete the needed variables for your project.

## Run Server

```bash
npm run dev
```

or

```bash
yarn dev
```
